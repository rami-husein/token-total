Token Total

A vanilla JavaScript BPE tokenizer for static websites
Count OpenAI tokens without a server

Why Token Total?

  • Zero dependencies - pure vanilla JavaScript
  • Works completely offline with browser caching
  • 100% compatible with tiktoken
  • Supports all OpenAI models (GPT-4o, GPT-4, GPT-3.5, etc.)
  • Perfect for static sites on GitHub Pages, Netlify, Vercel
import { encodingForModel } from './src/index.js'; const enc = await encodingForModel('gpt-4'); const tokens = enc.encode('Hello, world!'); // [9906, 11, 1917, 0]
